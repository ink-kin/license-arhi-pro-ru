Подготовительный период в кино включает в себя  
Разработка постановочного проекта фильма; Режиссёрский сценарий с экспликациями; Эскизы декораций, комбинированных съёмок, костюмов, реквизита, грима, чертежи и описания к ним; Съёмочные карты и операторские экспликации; Зарисовки кадров; Монтажно-технические разработки; Фотоматериалы, характеризующие изобразительную сторону картины; Звуковая экспликация фильма; Календарно-постановочный план; Генеральная смета; Выбор мест натурных съёмок; Подбор актёров и проведение предварительных репетиций; Заготовка постановочных средств; Утверждение постановочного проекта фильма.
