## Добро пожаловать в систему привлечения финансирования для кинопроектов

**ООО «Архи-про»** предоставляет комплексные услуги по привлечению средств для вашего кинопроекта через:

1. **Прямые переговоры** с корпорациями и брендами
2. **Юридическую упаковку** проекта
3. **Техническую интеграцию** с системами оплаты и ЭДО

### Наш подход

Мы создаём для вас готовую инфраструктуру, включающую:
- **Лендинг-оферту** с подключением эквайринга
- **Систему генерации договоров** в СБИС на основе шаблона
- **Прозрачную отчётность** через ЭДО
- **Юридическое сопровождение** всех сделок

### Как это работает

1. **Акцепт оферты** — вы знакомитесь с условиями и подписываете акцепт в СБИС
2. **Фиксированный платёж** — 45 000 ₽ за настройку всей системы
3. **Быстрая реализация** — в течение 50 дней получаете готовый инструмент
4. **Успешный гонорар** — платите только при привлечении средств (30% от лицензии, 10% от спонсорства)

### Открытость и прозрачность

- Все изменения оферты публикуются в репозитории GitHub
- Каждый акцепт фиксируется в ЭДО СБИС с юридической силой
- Отсутствие скрытых условий и схем
